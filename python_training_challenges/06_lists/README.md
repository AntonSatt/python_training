# Challenge 6: Lists

## The Problem
The `list_operations.py` has broken implementations for common list operations.

## Your Task
Fix the following functions:
1. `find_max` - should return the maximum value in a list
2. `remove_duplicates` - should return a list with duplicates removed
3. `flatten_list` - should flatten a nested list into a single list

## Expected Output
```
Max: 45
Unique: [1, 2, 3, 4, 5]
Flat: [1, 2, 3, 4, 5, 6]
```

## Run the tests
```bash
python test_list_operations.py
```
