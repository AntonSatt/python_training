# Challenge 5: Functions

## The Problem
The `string_utils.py` has broken function implementations for string manipulation.

## Your Task
Fix the following functions:
1. `reverse_string` - should return the reversed string
2. `is_palindrome` - should return True if string is a palindrome
3. `count_vowels` - should return the number of vowels in a string

## Expected Output
```
Reversed: dlrow olleh
Is palindrome: True
Vowels: 2
```

## Run the tests
```bash
python test_string_utils.py
```
