# Challenge 8: File I/O

## The Problem
The `file_handler.py` has broken file reading and writing operations.

## Your Task
Fix the following functions:
1. `read_file` - should read and return the contents of a file
2. `write_file` - should write content to a file
3. `count_lines` - should return the number of lines in a file

## Expected Output
```
Content: Hello World
Lines: 3
```

## Run the tests
```bash
python test_file_handler.py
```
