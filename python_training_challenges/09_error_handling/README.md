# Challenge 9: Error Handling

## The Problem
The `safe_calculator.py` has missing or incorrect error handling.

## Your Task
Fix the following functions to handle errors gracefully:
1. `safe_divide` - should handle division by zero
2. `safe_convert_to_int` - should handle invalid conversions
3. `safe_get_element` - should handle index out of range errors

## Expected Output
```
Division: Cannot divide by zero
Conversion: Invalid conversion
Index: Index out of range
```

## Run the tests
```bash
python test_safe_calculator.py
```
