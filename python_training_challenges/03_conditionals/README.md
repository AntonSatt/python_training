# Challenge 3: Conditionals

## The Problem
The `grade_calculator.py` has broken conditional logic for determining letter grades.

## Your Task
Fix the `get_letter_grade` function so it correctly returns:
- "A" for scores 90 and above
- "B" for scores 80-89
- "C" for scores 70-79
- "D" for scores 60-69
- "F" for scores below 60

## Expected Output
```
Score: 85, Grade: B
```

## Run the tests
```bash
python test_grade_calculator.py
```
