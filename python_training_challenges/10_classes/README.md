# Challenge 10: Classes and OOP

## The Problem
The `bank_account.py` has a broken BankAccount class implementation.

## Your Task
Fix the `BankAccount` class so it correctly:
1. Initializes with a balance (default to 0)
2. Implements `deposit` method that adds to balance
3. Implements `withdraw` method that subtracts from balance (no overdraft)
4. Implements `get_balance` method that returns current balance

## Expected Output
```
Balance: 150
After withdrawal: 100
```

## Run the tests
```bash
python test_bank_account.py
```
