# Challenge 7: Dictionaries

## The Problem
The `word_counter.py` has broken implementations for counting word frequencies.

## Your Task
Fix the `count_words` function so it correctly:
1. Counts the frequency of each word in a text
2. Ignores case (treats "The" and "the" as the same word)
3. Returns a dictionary with words as keys and counts as values

## Expected Output
```
{'the': 2, 'quick': 1, 'brown': 1, 'fox': 1}
```

## Run the tests
```bash
python test_word_counter.py
```
